Api Keys

This section describes what API keys are and how to use them.

Payculia authenticates your API requests using your account’s API key. If you do not include your key when making an API request or use one that is incorrect or outdated, Payculia will return an error.

We use one type of API key on Payculia: Secret Key

Credential Description
Secret Key This should be kept confidential and only stored on your own servers. Your account’s secret API key can perform any API request to Payculia without restriction. The secret key is used to authorize all your API calls on Payculia.
How to Obtain your API Key?

Your API keys are available on your Payculia Dashboard. Or you can find it below:

Authentication

Payculia uses OAuth 2.0 as the basic security protocol. To access payculia endpoints you'd need to use the secret key generated for you to access other endpoints.

With Payculia, you can authorize your API calls by your secret key. You send HTTP requests with the Authorization header that contains the word Basic followed by a space and the secret key as seen below.
Authorization: Basic Your-secret-key
